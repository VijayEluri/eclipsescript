EclipseScript Update Site

This is an Eclipse update site. To install the plug-in provided here:

  1. Open the Help menu.
  2. Select the Install New Software... menu item. A dialog will open.
  3. In the opened dialog, press the Add.. button to add this update site.
  4. Enter the following:
    • Name: EclipseScript
    • Location: https://eclipsescript.org/updates/
    and press the OK button.
